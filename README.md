# GTFS Query Portal

A GTFS-based spatial and spatio-temporal query portal for public transportation data using PostgreSQL, PostGIS, and Flask.

## Overview

This project focuses on importing GTFS data, storing it in a spatial database, running basic spatial and route-based queries, and visualizing the results through a web interface.

The first phase uses scheduled GTFS data. Real-time data may be added later if a suitable source becomes available.
